# Audio Assets Directory

Place your audio files here according to the specifications in `/AUDIO_ASSETS.md`.

## Required Files

1. `move.mp3` - Pour/splash sound (100-200ms)
2. `win_basic.mp3` - 1-star victory (500ms)
3. `win_good.mp3` - 2-star victory (700ms)
4. `win_perfect.mp3` - 3-star victory (1000ms)
5. `error.mp3` - Invalid move (50-100ms)
6. `undo.mp3` - Undo action (100-150ms)
7. `button_tap.mp3` - UI click (30-50ms)
8. `level_start.mp3` - Level begins (200-300ms)
9. `background.mp3` - Looping music (60-180s)

## Format

- **Format:** MP3
- **Sample Rate:** 44.1 kHz
- **Bit Rate:** 128 kbps
- **Normalized:** -3 dB to 0 dB (SFX), -6 dB to -3 dB (music)

## Free Resources

- **Freesound.org** - Community sounds
- **OpenGameArt.org** - Game assets
- **Incompetech.com** - Royalty-free music
- **JSFXR** - Generate simple game sounds

See `/AUDIO_ASSETS.md` for complete specifications and resource links.

## Testing

The app works without these files - they're optional. Add them when ready for audio.
